# Conferatur - 파일/폴더 비교 도구

[한국어](README.md) | [English](README.en.md)

Python tkinter 기반의 직관적인 파일 및 폴더 비교 도구입니다.

## ✨ 새로운 기능 (최신 업데이트)

- 🎨 **Notion 스타일 디자인 시스템** 적용 - 정돈된 톤의 모던 UI, PretendardStd 기본 폰트
- 🌐 **영어 UI 전환 지원** - 설정 메뉴에서 한국어/English를 즉시 전환하고 선택 언어를 자동 저장
- 🌳 **폴더 트리 강화** - 상태별 색상 태그(diff/left_only/right_only), 컬럼 정렬, 차이 개수 배지, ⊞ 모두 펼치기 / ⊟ 모두 접기 버튼
- 🔎 **폴더 파일 미리보기 확대** - 폴더 비교 결과에서 좌/우 파일 내용을 더 넓은 영역으로 확인
- 📄 **파일 비교 강화** - 라인 단위 hunk 배경 하이라이트(추가/삭제/변경 구분), 상태 바, ◀ ▶ 차이점 네비게이션 버튼
- 🗑️ **히스토리 삭제 발견성 개선** - 좌측 독립 destructive 버튼, 트리 우클릭 컨텍스트 메뉴, `<Delete>` 키 지원
- 💻 **macOS 완벽 지원** - Command 키 단축키 (Cmd+C/V/X/A)
- 📋 **클립보드 작업** - 복사/붙여넣기/잘라내기/전체선택
- 🖱️ **우클릭 컨텍스트 메뉴** - 모든 텍스트 영역에서 사용 가능
- 🔄 **완벽한 스크롤 동기화** - 마우스 휠, 스크롤바 드래그 모두 지원

## 주요 기능

### 1. 폴더 비교 모드
- **MD5 체크섬 비교**: 파일 내용의 정확한 비교
- **날짜 비교**: 파일 수정일 기반 비교
- **복합 비교**: MD5 + 날짜 비교
- **트리 형태 시각화**: 폴더 구조를 계층적으로 표시
- **상태별 색상 태그**: diff(차이) / left_only / right_only를 트리에서 색상으로 즉시 구분
- **컬럼 정렬**: 상태/크기/수정일 컬럼 헤더 클릭으로 정렬 (오름/내림차순 토글)
- **차이 개수 배지**: 폴더 노드에 하위 차이 항목 개수 표시
- **모두 펼치기/접기**: ⊞ / ⊟ 버튼으로 트리 전체를 한 번에 토글
- **제외 패턴**: `.gitignore` 스타일 패턴으로 폴더/파일 제외 (`node_modules/`, `*.pyc` 등)
- 양방향 파일 복사 (왼쪽 ↔ 오른쪽)
- 선택한 파일 삭제 기능
- **확장된 파일 내용 미리보기**: 목록에서 파일 선택 시 더 넓은 좌/우 패널에 내용을 표시하고 차이점 하이라이트
- **완벽한 스크롤 동기화**: 미리보기에서 마우스 휠, 스크롤바 드래그 모두 동기화
- **히스토리 및 즐겨찾기 지원**

### 2. 텍스트 직접 비교 모드
- 두 개의 텍스트를 직접 입력하여 비교
- **문자 단위 상세 비교**: 라인 내에서 정확히 어느 문자가 다른지 표시
- 차이점을 연한 붉은색으로 하이라이트
- 양방향 텍스트 적용 기능
- 실시간 편집 및 비교
- **복사/붙여넣기**: Cmd+C/V (macOS) 또는 Ctrl+C/V (Windows/Linux)
- **우클릭 메뉴**: 복사, 잘라내기, 붙여넣기, 전체 선택
- **완벽한 스크롤 동기화**: 마우스 휠, 스크롤바 드래그 모두 동기화
- **히스토리 및 즐겨찾기 지원**

### 3. 파일 내용 비교 모드
- 두 파일의 내용을 불러와서 비교
- **문자 단위 상세 비교**: 라인 내에서 정확히 어느 문자가 다른지 표시
- **라인 단위 hunk 배경 하이라이트**: 추가/삭제/변경된 라인을 배경색으로 구분 표시
- **상태 바**: 총 차이 개수와 현재 위치를 하단에서 실시간 확인
- **차이점 네비게이션**: ◀ 이전 / ▶ 다음 버튼으로 차이점 간 빠르게 이동
- 내용 수정 후 저장 가능
- UTF-8 인코딩 지원
- **복사/붙여넣기**: Cmd+C/V (macOS) 또는 Ctrl+C/V (Windows/Linux)
- **우클릭 메뉴**: 복사, 잘라내기, 붙여넣기, 전체 선택, 블록 단위 좌/우 복사
- **완벽한 스크롤 동기화**: 마우스 휠, 스크롤바 드래그 모두 동기화
- **히스토리 및 즐겨찾기 지원**

### 4. 히스토리 및 즐겨찾기 기능
- **자동 히스토리 저장**: 최근 20개의 비교 작업 자동 저장
- **즐겨찾기 관리**: 자주 사용하는 비교 설정을 이름을 지정하여 저장
- **즐겨찾기 이름 변경**: 저장된 즐겨찾기의 이름 수정 가능
- **히스토리/즐겨찾기 삭제**: 세 가지 방법 지원
  - 좌측 독립 `🗑️ 선택 항목 삭제` 버튼 (눈에 띄는 destructive 색상)
  - 트리 항목 우클릭 → 컨텍스트 메뉴
  - 항목 선택 후 `Delete` 키
- **검색 필터**: 다이얼로그 상단 검색창으로 히스토리/즐겨찾기 항목 즉시 필터링
- **영구 저장**: `~/.conferatur/config.json`에 저장되어 프로그램 종료 후에도 유지

### 5. 언어, 폰트 및 디자인 설정
- **UI 언어 전환**: 메뉴 → 설정 → 언어에서 한국어/English 즉시 전환
- **언어 설정 저장**: 선택한 언어는 `~/.conferatur/config.json`에 저장되어 다음 실행 때 유지
- **PretendardStd 기본 폰트**: 한글/영문 모두 깔끔하게 표시
- **폰트 변경**: 메뉴 → 설정 → 폰트 설정에서 16가지 프리셋 폰트 중 선택
- **폰트 크기 조절**: 8~20pt 범위에서 실시간 미리보기 후 일괄 적용
- **Notion 톤 디자인 시스템**: 차분한 색상 팔레트와 일관된 위젯 스타일

## 설치 요구사항

- Python 3.7 이상
- tkinter (Python 기본 내장)
- **ttkbootstrap** (필수)

### 의존성 설치

```bash
pip install ttkbootstrap
```

또는 pip3를 사용하는 경우:
```bash
pip3 install ttkbootstrap
```

### tkinter 설치 확인

```bash
python3 -c "import tkinter; print('tkinter 사용 가능')"
```

Ubuntu/Debian에서 tkinter가 없는 경우:
```bash
sudo apt-get install python3-tk
```

macOS에서:
```bash
# Python이 Homebrew로 설치된 경우 tkinter는 자동으로 포함됨
brew install python-tk@3.11  # Python 버전에 맞게 조정
```

- 복사
- 잘라내기
- 붙여넣기

### UI 언어 변경

1. 상단 메뉴 → "설정" → "언어" 선택
2. "한국어" 또는 "English" 선택
3. 현재 입력된 폴더/파일 경로와 텍스트 내용은 유지한 채 UI 문구가 즉시 갱신됩니다
4. 선택한 언어는 자동 저장되어 다음 실행 시에도 유지됩니다

### 1. 폴더 비교

1. "폴더 비교" 탭 선택
2. 왼쪽/오른쪽 폴더 선택
   - **또는** "📜 히스토리에서 불러오기" 클릭하여 이전 비교 불러오기
   - **또는** "⭐ 즐겨찾기에서 불러오기" 클릭하여 저장된 비교 불러오기
3. 비교 방법 선택:
   - **MD5 비교**: 파일 내용이 다른지 정확히 비교
   - **날짜 비교**: 파일 수정일이 다른지 비교
   - **MD5 + 날짜**: 두 가지 모두 확인
4. "비교 시작" 버튼 클릭 (자동으로 히스토리에 저장됨)
5. 결과를 트리 구조로 확인
   - 📁 아이콘: 폴더 (클릭하여 펼치기/접기, 또는 상단 ⊞/⊟ 버튼으로 일괄 토글)
   - 상태별 색상으로 차이/한쪽만 존재하는 항목 즉시 식별
   - 폴더 노드의 차이 개수 배지로 변경 범위 파악
   - 컬럼 헤더 클릭으로 상태/크기/수정일 정렬
   - 파일 선택 시 하단의 확장된 좌/우 패널에서 내용 미리보기
6. 필요시 파일 복사 또는 삭제
7. 자주 사용하는 비교는 "⭐ 즐겨찾기에 추가" 버튼으로 저장

### 2. 텍스트 비교

1. "텍스트 비교" 탭 선택
2. 왼쪽/오른쪽 텍스트 영역에 내용 입력
   - 키보드 단축키나 우클릭 메뉴로 복사/붙여넣기 가능
   - **또는** "📜 히스토리에서 불러오기" / "⭐ 즐겨찾기에서 불러오기"
3. "비교하기" 버튼 클릭 (자동으로 히스토리에 저장됨)
4. 차이점 확인 (연한 붉은색으로 표시)
5. 필요시 한쪽으로 적용
6. 자주 사용하는 텍스트는 즐겨찾기에 추가

### 3. 파일 비교

1. "파일 비교" 탭 선택
2. 왼쪽/오른쪽 파일 선택
   - **또는** "📜 히스토리에서 불러오기" / "⭐ 즐겨찾기에서 불러오기"
3. "비교하기" 버튼 클릭 (자동으로 히스토리에 저장됨)
4. 차이점 확인 및 수정
   - 라인 단위 hunk 배경 색으로 추가/삭제/변경 즉시 식별
   - 하단 상태 바에서 총 차이 개수와 현재 위치 확인
   - ◀ 이전 / ▶ 다음 버튼으로 차이점 간 이동
   - 텍스트 영역에서 복사/붙여넣기, 우클릭 메뉴로 블록 단위 좌/우 복사 가능
5. 필요시 파일 저장
6. 자주 사용하는 파일 쌍은 즐겨찾기에 추가

### 4. 히스토리 및 즐겨찾기 관리

#### 히스토리 관리
1. 상단 메뉴 → "히스토리" → 원하는 모드(폴더/파일/텍스트) 선택
2. 다이얼로그 상단 검색창으로 필터링 가능
3. 항목 삭제 (세 가지 방법):
   - 좌측 하단 **🗑️ 선택 항목 삭제** 버튼 클릭
   - 항목 **우클릭** → 컨텍스트 메뉴에서 삭제
   - 항목 선택 후 **Delete** 키

#### 즐겨찾기 관리
1. 상단 메뉴 → "즐겨찾기" → 원하는 모드 선택
2. 항목 선택 후:
   - **이름 변경** 버튼: 즐겨찾기 이름 수정
   - **🗑️ 선택 항목 삭제** 버튼 / 우클릭 / `Delete` 키: 제거

## 주요 기능 설명

### 비교 방법

- **MD5 비교**: 파일의 해시값을 비교하여 내용이 정확히 같은지 확인합니다. 파일 이름이나 날짜가 달라도 내용이 같으면 동일한 것으로 판단합니다.
- **날짜 비교**: 파일의 마지막 수정 시간을 비교합니다. 어느 쪽이 더 최신인지 확인할 수 있습니다.
- **MD5 + 날짜**: 두 방법을 모두 사용하여 내용의 차이와 날짜 차이를 함께 확인합니다.

### 파일 상태

- **동일**: 파일이 완전히 같음 (표시되지 않음)
- **내용 다름 (MD5)**: MD5 해시가 다름
- **왼쪽만 존재**: 왼쪽 폴더에만 파일이 있음
- **오른쪽만 존재**: 오른쪽 폴더에만 파일이 있음
- **왼쪽이 최신**: 왼쪽 파일이 더 최근에 수정됨
- **오른쪽이 최신**: 오른쪽 파일이 더 최근에 수정됨

### 복사 기능

- **왼쪽 → 오른쪽**: 선택한 파일을 왼쪽에서 오른쪽으로 복사
- **오른쪽 → 왼쪽**: 선택한 파일을 오른쪽에서 왼쪽으로 복사
- 복사 시 디렉토리 구조가 자동으로 생성됩니다
- 파일이 이미 존재하면 덮어씁니다

### 스크롤 동기화

텍스트 비교 및 파일 미리보기에서 양쪽 패널의 스크롤이 자동으로 동기화됩니다:
- **마우스 휠**: 한쪽에서 스크롤하면 다른쪽도 같이 스크롤
- **스크롤바 드래그**: 스크롤바를 드래그해도 양쪽이 동기화

### 히스토리 및 즐겨찾기

#### 히스토리
- 모든 비교 작업은 자동으로 히스토리에 저장됩니다
- 최근 20개의 작업만 유지됩니다 (오래된 것은 자동 삭제)
- 각 모드(폴더/파일/텍스트)별로 별도 관리
- 타임스탬프와 함께 저장되어 언제 비교했는지 확인 가능

#### 즐겨찾기
- 자주 사용하는 비교 설정을 이름과 함께 저장
- 개수 제한 없음
- 즐겨찾기 이름 변경 가능
- 각 모드별로 독립적으로 관리
- 예시:
  - 폴더: "프로젝트 백업 비교", "소스 vs 빌드" 등
  - 파일: "설정 파일 비교", "버전 1 vs 버전 2" 등
  - 텍스트: "템플릿 비교", "번역 검토" 등

#### 데이터 저장 위치
- 모든 히스토리, 즐겨찾기, 언어/폰트 설정, 제외 패턴은 `~/.conferatur/config.json` 파일에 저장
- 프로그램을 종료하고 다시 실행해도 데이터 유지
- JSON 형식으로 저장되어 필요시 수동 편집 가능

## UI 테마

이 프로그램은 **Notion 스타일 디자인 시스템**(`design.md`)을 기반으로 자체 위젯 스타일과 색상 팔레트를 사용합니다:
- 🎨 차분한 톤의 색상 팔레트 (peach/rose/mint/lavender 등 의미별 강조)
- ✍️ **PretendardStd** 기본 폰트로 한국어/English UI 모두 깔끔하게 표시
- 📐 일관된 버튼 역할(primary / secondary / destructive / ghost) 정의
- 🖼️ 라인 hunk 배경, 트리 상태 태그 등 모든 시각 요소가 동일한 팔레트에서 파생

`ttkbootstrap`은 기반 위젯 프레임워크로 사용되며, 추가 커스텀 스타일이 `compare_tool.py`에 적용되어 있습니다.
